Why immerser?

Nowadays designers tends to create complex logic and fix parts of interface. Also they colour page sections contrastly. How to deal with that mess? Immerser can help you.

Immerser don't have any dependencies and written on vanilla js. It fast, because it calculate states once on init and then watch scroll position and schedule redraw DOM in next event loop tick with requestAnimationFrame. Script change transform property, so it uses graphic hardware acceleration.
